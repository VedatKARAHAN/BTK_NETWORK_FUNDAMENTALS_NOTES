| Kategori                      | Terim                          | Açıklama                                                                                         |
| :---------------------------- | :----------------------------- | :----------------------------------------------------------------------------------------------- |
| **Ağ Türleri**                | **Internet**                   | Dünya genelindeki birbirine bağlı tüm ağların oluşturduğu küresel sistem.                        |
|                               | **Small home network**         | Bir evdeki az sayıda cihazı birbirine ve internete bağlayan basit ağ.                            |
|                               | **SOHO network**               | **Küçük Ofis/Ev Ofis** ortamında kullanılan, birden fazla kullanıcıyı destekleyen ağ.            |
|                               | **Medium to large network**    | Kurumsal veya büyük ölçekli bir organizasyonu kapsayan karmaşık ağ.                              |
| **Uç Cihazlar (End Devices)** | **Smartphone**                 | İnternet erişimi ve uygulama yeteneği olan akıllı cep telefonu.                                  |
|                               | **Tablet**                     | Taşınabilir, dokunmatik ekranlı kişisel bilgisayar.                                              |
|                               | **Smartwatch**                 | Bileğe takılan, bildirim ve sağlık takibi yapabilen cihaz.                                       |
|                               | **Smartglasses**               | Artırılmış gerçeklik veya bilgi gösterebilen akıllı gözlük.                                      |
|                               | **Connected home devices**     | İnternete bağlı, ev otomasyonu sağlayan cihazlar (örn: akıllı ampul).                            |
|                               | **Smart car**                  | İnternet bağlantılı, gelişmiş sensör ve sistemlere sahip otomobil.                               |
| **IoT Bileşenleri**           | **RFID tag**                   | Radyo frekansı ile nesneleri tanımlayan etiket (örn: temassız kartlar).                          |
|                               | **Sensor**                     | Fiziksel bir olguyu (sıcaklık, basınç vb.) ölçen ve veriyi yakalayan cihaz.                      |
|                               | **Actuator**                   | Elektrik sinyalini alarak fiziksel bir eyleme (hareket, vana açma vb.) dönüştüren cihaz.         |
| **Veri Türleri**              | **Volunteered data**           | Kullanıcı tarafından bilerek ve isteyerek sağlanan veriler (örn: anket yanıtı).                  |
|                               | **Observed data**              | Sensörler veya izleme sistemleri aracılığıyla otomatik olarak toplanan ham veriler.              |
|                               | **Inferred data**              | Gözlemlenen verilerin analizi sonucunda elde edilen, çıkarım yoluyla üretilen bilgiler.          |
| **Temel Veri Birimi**         | **Bit**                        | Bilgisayardaki en küçük bilgi birimi (0 veya 1).                                                 |
|                               | **ASCII**                      | İngilizce karakterleri ve diğer metinleri temsil etmek için kullanılan standart kodlama sistemi. |
| **İletim Ortamı (Media)**     | **Media**                      | Verinin kaynağından hedefine iletildiği fiziksel yol (kablo veya kablosuz).                      |
|                               | **Electrical signal**          | Bakır kablolar üzerinden elektrik akımı ile iletilen sinyal.                                     |
|                               | **Optical signal**             | Fiber optik kablolar üzerinden ışık ile iletilen sinyal.                                         |
|                               | **Wireless signal**            | Havadan radyo dalgaları aracılığıyla yayılan sinyal.                                             |
|                               | **LAN Media**                  | Yerel Ağ (LAN) içinde kullanılan kablo türleri (örn: Ethernet kablosu).                          |
|                               | **WAN Media**                  | Geniş Alan Ağı (WAN) için kullanılan uzun mesafeli bağlantı ortamları.                           |
| **Hız ve Kapasite**           | **Bandwidth**                  | Bir bağlantının belirli bir zamanda taşıyabileceği maksimum veri miktarı (potansiyel hız).       |
|                               | **Throughput**                 | Gerçek bir zaman diliminde başarıyla iletilen ve işlenen veri miktarı (gerçekleşen hız).         |
|                               | **Bits per second (bps)**      | Saniyede iletilen bit sayısı (temel hız birimi).                                                 |
|                               | **Kilobits per second (kbps)** | Saniyede bin (kilo) bit hız.                                                                     |
|                               | **Megabits per second (Mbps)** | Saniyede milyon (mega) bit hız.                                                                  |
|                               | **Gigabits per second (Gbps)** | Saniyede milyar (giga) bit hız.                                                                  |
|                               | **Terabits per second (Tbps)** | Saniyede trilyon (tera) bit hız.                                                                 |
| **Ağ Rolleri ve Mimarileri**  | **Client**                     | Sunucudan hizmet veya kaynak talep eden cihaz (örn: PC, tarayıcı).                               |
|                               | **Server**                     | İstemcilere hizmet veya kaynak sağlayan cihaz (merkezi kaynak).                                  |

|
| | **Email server** | E-posta gönderimi, alımı ve depolaması için hizmet sağlayan sunucu. |
| | **Web server** | Web sayfalarını barındıran ve istemcilere HTTP protokolü ile sunan sunucu. |
| | **File server** | Dosyaları merkezi olarak depolayan ve ağ üzerinde paylaşan sunucu. |
| | **Peer-to-peer (P2P) network** | Cihazların hem istemci hem de sunucu olarak davrandığı merkezi olmayan ağ. |
| | **Peer-to-peer applications** | İki veya daha fazla kullanıcının doğrudan birbirine bağlandığı uygulamalar (örn: dosya paylaşım programları). |
| **Ağ Cihazları (Intermediate)** | **End device** | İletişimin başladığı veya bittiği cihaz (örn: bilgisayar, telefon). |
| | **Intermediate (intermediary) Device** | Veri akışını yönlendiren, filtreleyen veya yöneten ağ cihazı (örn: router, switch). |
| | **Device** | Genel olarak ağa bağlanabilen her türlü ekipman. |
| | **Network media** | İletişimi sağlayan fiziksel ortam (kablolar veya kablosuz). |
| | **Wireless router** | Kablosuz erişim noktası sağlayıp ağ trafiğini yönlendiren cihaz. |
| | **LAN switch** | Yerel ağ (LAN) içindeki cihazları aynı ağda birbirine bağlayan cihaz. |
| | **Router** | Farklı ağlar (LAN'lar arası veya LAN-WAN arası) arasındaki trafiği yönlendiren cihaz. |
| | **Multilayer switch** | Hem anahtarlama (Layer 2) hem de yönlendirme (Layer 3) yapabilen gelişmiş anahtar. |
| | **Firewall appliance** | Ağ güvenliğini sağlayan, gelen ve giden trafiği kurallara göre filtreleyen özel cihaz.
